# IpTracker-using-python-

Description
A Python GUI app that tracks geolocation info from any IP address — shows city, region, country, ISP, coordinates and timezone.


Requirements
-- Python 3.x
-- requests library

Installation
py -m pip install requests

how to run 

py ip_tracker.py 

Type above code on terminal to run


How to Use
Enter an IP address in the input field
Click Track
Shows the Result or error in case of an issue 
Click Clear to reset

API
Data provided by ip-api.com — free, no API key required.
